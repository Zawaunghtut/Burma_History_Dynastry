🇲🇲 မြန်မာ့မင်းဆက်များ ရာဇဝင်အကျဉ်း (Burma History Finder)
မြန်မာ့သမိုင်းကို စိတ်ဝင်စားသူများ၊ ကျောင်းသား/သူများနှင့် ဗဟုသုတရှာဖွေသူများအတွက် မြန်မာမင်းဆက်အသီးသီးကို တစ်နေရာတည်းတွင် အလွယ်တကူ ရှာဖွေလေ့လာနိုင်ရန် ဖန်တီးထားသော Single-Page Web Application ဖြစ်သည်။
✨ ထူးခြားချက်များ
 * မင်းဆက်စုံလင်မှု: တကောင်းခေတ်မှသည် ကုန်းဘောင်ခေတ်အထိ မင်းဆက်ပေါင်းများစွာကို စုစည်းဖော်ပြထားသည်။
 * မြန်ဆန်သော ရှာဖွေမှုစနစ်: Search Box တွင် မင်းအမည် သို့မဟုတ် ခေတ်အမည်ကို ရိုက်ထည့်ရုံဖြင့် အလွယ်တကူ ရှာဖွေနိုင်သည်။
 * Responsive Design: ဖုန်း၊ Tablet၊ ကွန်ပျူတာ အားလုံးတွင် ကြည့်ရှုရ အဆင်ပြေအောင် ဖန်တီးထားသည်။
 * Single File: HTML, CSS, JavaScript အားလုံးကို ဖိုင်တစ်ခုတည်း (Single Code) ဖြင့် စုစည်းထားသဖြင့် အသုံးပြုရ အလွန်လွယ်ကူသည်။
📂 ပါဝင်သော မင်းဆက်များ
၁။ ပထမတကောင်းမင်းဆက် (၃၃ ဆက်)
၂။ ဒုတိယတကောင်းမင်းဆက် (၁၇ ဆက်)
၃။ သရေခေတ္တရာမင်းဆက် (၂၇ ဆက်)
၄။ ပုဂံမင်းဆက် (၅၈ ဆက်)
၅။ မဂဒူး (ဝါရီရူး) မင်းဆက် (၁၈ ဆက်)
၆။ အင်းဝမင်းဆက် (၃၀ ဆက်)
၇။ တောင်ငူမင်းဆက် (၈ ဆက်)
၈။ ကုန်းဘောင်မင်းဆက် (၁၁ ဆက်)
🚀 အသုံးပြုနည်း
၁။ Myanmar_Dunasty.html ဖိုင်ကို Download ဆွဲပါ။
၂။ မိမိနှစ်သက်ရာ Browser (Chrome, Firefox, Safari, Edge) တစ်ခုခုဖြင့် ဖွင့်ပါ။
၃။ ရှာဖွေလိုသော အချက်အလက်ကို Search Box တွင် ရိုက်ထည့်၍ ရှာဖွေပါ။
🛠 နည်းပညာ
 * HTML5 - ဖွဲ့စည်းတည်ဆောက်ပုံ
 * CSS3 - အလှဆင်ခြင်းနှင့် Responsive ဖြစ်စေခြင်း
 * JavaScript (Vanilla) - ရှာဖွေမှုစနစ်နှင့် အပြန်အလှန်သက်ရောက်မှု
